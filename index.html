    About Clarity

    clarity is a mobile and web application geared towards closing the student-teacher feedback loop. clarity aims to be an application that disrupts the traditional model of student-teacher interaction by providing a channel for live and direction interaction between the students themsevles and thier teacher. clarity provides crowd-sourced content that the teacher can use to answer the most pressing questions at any point in the lecture.
